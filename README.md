# Python File Organizer

## Description
Organizes files in a folder into categories like Images, Documents, Audio, Videos, and Compressed Files. Unmatched files go into a "others" folder.

##Features
-Categorizes files based on extensions
- Creates folders automatically
- Moves files into correct folders


## How to Run
1. Run the script: python file_organizer.py
2. Enter the folder path when prompted
